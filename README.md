# Chippewa Valley Bin Cleaning Website

A modern, responsive marketing website for Chippewa Valley Bin Cleaning, styled to match the company logo's blue-and-green eco-friendly aesthetic.

## Quick Start

Open `index.html` in your browser, or run a local server:

```bash
npx serve .
```

Then visit `http://localhost:3000`.

## Customize

- **Contact info**: Update phone, email, and service areas in `index.html`
- **Pricing**: Edit plan prices and details in the `#pricing` section
- **Colors**: Adjust CSS variables at the top of `styles.css`

## Files

- `index.html` — Main page with all sections
- `styles.css` — Brand styling and responsive layout
- `script.js` — Mobile nav, scroll effects, form handling
- `assets/logo.png` — Company logo
